# Business Card Scanner

A simple web app that scans business cards using your browser's camera or uploaded images.

## Features
- Upload images of business cards
- Extract text using OCR (Tesseract.js)
- Save data in your browser's storage
- Export to Excel
- Search through scanned cards
- Works on any device with a browser

## How to Use
1. Open `index.html` in any browser
2. Upload front image of business card (required)
3. Upload back image (optional)
4. Click "Scan Card"
5. Data will be extracted and displayed
6. Use "Save to Browser" to save data
7. Use "Export to Excel" to download data

## Access from Any Device
Just open this link in any browser:
`https://YOUR_USERNAME.github.io/business-card-scanner/`

## Note
- Data is saved in your browser's local storage
- Different browsers/devices have separate storage
- To sync across devices, export Excel file and import elsewhere
